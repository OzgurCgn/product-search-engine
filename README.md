# Product Search Engine

A product search prototype designed to deliver fast and highly relevant results by utilizing **TF-IDF embeddings** and a **Qdrant** vector database. 

## 🚀 Overview
This project focuses on modern information retrieval techniques, combining traditional text processing (TF-IDF) with advanced vector search capabilities. The goal is to build a robust foundation for a scalable product search engine.

## 🛠️ Tech Stack
* **Language:** Python
* **Vector Database:** Qdrant
* **Algorithm:** TF-IDF (Term Frequency-Inverse Document Frequency)

## 🚧 Current Status & Future Roadmap
This project is currently **in progress**. Ongoing developments include:
* **Dataset Expansion:** Scaling the system to handle larger product datasets.
* **Payload Filtering:** Implementing advanced filtering mechanisms within Qdrant to refine search results based on specific product attributes.
* **UI Improvements:** Developing a cleaner and more intuitive user interface for end-users to interact with the search engine.
